✨ Features
Total Metrics: Real-time calculation of total stock quantity and inventory valuation.

Dynamic Inventory List: Clean tabular view of SKUs, item names, quantities, and individual price points.

Management Tools: Integrated form to add new stock or edit existing entries with ease.

Minimalist UI: High-contrast dark mode designed for focus and modern desktop environments.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Grid/Flexbox styling)

Logic: JavaScript (ES6+) for dynamic calculations and DOM manipulation
